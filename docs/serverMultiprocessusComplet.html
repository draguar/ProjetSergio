serverMultiprocessusComplet module¶

serverMultiprocessusComplet.check_passwd(fpasswd, ID, passwd)[source]¶

Parcours le fichier fpasswd et regarde si le passwd correspond au mot de passe de l’utilisateur ID

Paramètres:
  • fpasswd (str) – nom du fichier à parcourir
  • ID (str) – nom de l’utilisateur dont on check le mot de passe
Retourne:

True si le mot de passe est le bon, false sinon
Type retourné:

bool

Example:

>>> check_passwd("run/passwd.txt","test","test")
